# 🤖 بوت المتجر الرقمي — Digital Store Bot

## 📁 هيكل الملفات
```
├── bot.py          ← الكود الرئيسي للبوت
├── database.py     ← قاعدة البيانات
├── config.py       ← الإعدادات (التوكن + الأدمن)
├── requirements.txt← المكتبات المطلوبة
└── store.db        ← قاعدة البيانات (بتتعمل تلقائي)
```

---

## ⚙️ خطوات التشغيل

### 1. تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 2. إعداد config.py
افتح ملف `config.py` وعدّل:
- `BOT_TOKEN` ← التوكن من @BotFather
- `ADMIN_IDS`  ← الـ ID بتاعك (من @userinfobot)

### 3. تشغيل البوت
```bash
python bot.py
```

## 🏪 هيكل المتجر

```
🛍️ المنتجات
  └── 📂 كاتيجري (مثال: متابعين)
        └── 📱 تطبيق (مثال: Instagram)
              ├── خدمة 1 (مثال: 1000 متابع — $5)
              └── خدمة 2 (مثال: 5000 متابع — $20)
        └── 📱 تطبيق (مثال: TikTok)
              └── ...
  └── 📂 كاتيجري (مثال: لايكات)
        └── ...
```

---

## 👤 للمستخدم
| الزر | الوظيفة |
|------|---------|
| 🛍️ المنتجات | تصفح الكاتيجريز والخدمات |
| 💼 محفظتي | عرض الرصيد وآخر المعاملات |
| ➕ شحن رصيد | إرسال طلب إيداع مع إيصال |
| 📋 طلباتي | عرض سجل المشتريات |

## ⚙️ للأدمن
| الزر | الوظيفة |
|------|---------|
| 📂 الكاتيجريز | إضافة/حذف كاتيجريز وتطبيقات وخدمات |
| 📦 المنتجات | عرض كل الخدمات |
| 👥 المستخدمين | عرض كل المستخدمين والأرصدة |
| 💳 الإيداعات | مراجعة وقبول/رفض طلبات الشحن |
| 📊 إحصائيات | إحصائيات عامة للمتجر |

---

## 💡 ملاحظات مهمة
- لو عندك `store.db` قديم احذفه قبل التشغيل
- البوت بيشتغل بـ SQLite (مش محتاج أي سيرفر خارجي)
- كل البيانات بتتحفظ في ملف `store.db` تلقائياً
